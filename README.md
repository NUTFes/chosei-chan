このリポジトリはハッカソンで作成した「調整ちゃん」のリポジトリです。

## Getting Started

```bash
npm install
npm run dev
```

## Storybook Usage

以下のコマンドでポート6006番にStorybookが起動します

```bash
npm run storybook
```

## ルール

- ブランチ名
  - `{prefix}/{name}/{issueNo}`
  - ex: feat/imaimai/25
- コミットメッセージ
  - `[{prefix}] {commit message}`
  - ex: [fix] ボタンコンポーネントバグの修正

- prefix
  - feat: 新規開発
  - fix: バグ修正
